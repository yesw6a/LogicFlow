# 0.4.0

# @logicflow/core

  • 支持新的组件注册方式；
import { RectNode, RectModel } from '@logicflow/core'

lf.register({
  type: 'square',
  view: RectNode,
  model: RectModel,
});
  • 支持class作为插件

# @logicflow/extension

  • 新增：ResizeNode插件（ResizeNode）。
  • 新增：bpmn插件增加路径生成和自动布局功能。
  • 新增: 节点拖入折线插件。

# 兼容性

  • 新增：兼容IE11
